Integraciones básicas 
Las distintas soluciones para vendedores que ofrece DineroMail by PayU pueden crearse directamente a través del Asistente. Para ello, deberás ingresar a tu cuenta de DineroMail, ir a la sección de Soluciones para Vendedores, elegir la solución que se adapte a tus necesidades y siguiendo los pasos indicados podrás comenzar a cobrar en minutos. 
No necesitas conocimientos técnicos. ¡Es tán fácil como copiar y pegar! 
 
Integraciones avanzadas 
Si estás familiarizado con la codificación HTML, puedes escribir personalmente el código para tus Botones de Pago y Carrito de Compra

Con un poco más de conocimiento podrás hacer integraciones un poco más avanzadas tales como:
  1. Carrito de Compras
  2. Botón de Pago
  3. Botón de Pago con Montos Variables
  4. Iframe para Tarjeta de Crédito

  6. One Click To Buy para Tarjeta de Crédito

API  
Para las integraciones más complejas, DineroMail by PayU desarrolló un conjunto de APIs para que los programadores puedan integrar DineroMail a cualquier sistema. A través de estas APIs de DineroMail by PayU podrás crear botones de pago, carritos de compras, notificaciones de pago, verificar el balance de tu cuenta, realizar pagos, generar códigos de barra y otras funcionalidades. 
 
La API de Dineromail by PayU pone una potente funcionalidad a disposición de los programadores que a través de una interfase sólida y flexible podrán:
  1. Consultar Saldo de su Cuenta de DineroMail
  2. Generar Códigos de Barra
  3. Consulta de Estado de Operaciones
  4. Enviar Dinero
  5. Retirar Fondos
  6. Tarjeta de Crédito (Solo para empresas PCI)

IPN (instant payment notification)  
Sistema que muestra la información de los movimientos de tu cuenta de DineroMail en un lenguaje universal denominado XML.  
Existen dos versiones para este sistema 
 
IPN1: Mediante una consulta, obtendrás la información de las transacciones completas que ha han sido acreditadas en tu cuenta DineroMail by PayU. Podrás ver aquellos eventos exitosos o completos dentro de tu cuenta, ya sea transferencias a tu cuenta bancaria, pagos, créditos, ventas, etc. 
 
IPN2: Obtendrás la información de todas las transacciones realizadas a través de la plataforma de PayU. Podrás ver aquellos eventos completos, cancelados o pendientes dentro de tu cuenta. 
La esencia del IPN V.2 es darte por notificado de un nuevo evento en tu cuenta. En base a la información que recibas podrás programar diferentes acciones especificas a tus necesidades. Ejemplos del tipo de acciones que podrás tomar cuando recibas un mensaje pueden ser las siguientes: 
  1. Provocar ordenes de envío automático de correo electrónico
  2. Actualizar tus registros de venta
  3. Actualizar tu lista de clientes
  4. Dar de alta a un cliente en tu base de datos
  5. Permitirle el acceso a un determinado sistema
  6. Notificar el servicio de logística de un nuevo envío
