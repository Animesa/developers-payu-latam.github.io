Integraciones PayU.

Elige el método que más se ajuste a las necesidades de tu negocio para integrar tu sitio web.

API

Diseña tu propio checkout y administra el sistema de pagos de tu sitio web.

Disfruta servicios adicionales como Tokenización y Pagos recurrentes.

Tu tienda se conecta con la plataforma PayU, pero el proceso de pago es manejado desde tu sitio.

Conocer más

SDK

Acceso a los beneficios de la integración API, aún más ágil.

Disponible para los lenguajes JAVA y PHP.

Diseño personalizado de tu checkout para que tu cliente tenga una mejor experiencia de compra.

Conocer más

Web Checkout

PayU administra el sistema de pagos.

Integra el servicio de manera sencila, con una implementación más eficaz.

PayU captura los datos de las tarjetas de crédito de tus clientes, por lo que es más fácil obtener la certificación PCI DSS
