Integraciones PayU.

Elige el método para integrar tu sitio web con PayU que más se ajuste a las necesidades de tu negocio.

Integración API

Diseña tu propio checkout y administra el sistema de pagos de tu sitio web.

Accede a servicios adicionales como tokenización o pagos recurrentes.

El cliente permanecerá en tu sitio web durante la compra, tu marca será la protagonista.

Conocer más

Integración SDK

Accede a los beneficios de la API PayU, con una integración más ágil.

Disponible para los lenguajes JAVA y PHP.

Diseña y personaliza tu propio checkout , tu cliente permanecerá en tu sitio web durante la compra.

Conocer más

Integración WebCheckout

PayU administra el sistema de pagos.

Integra el servicio de manera más sencilla, reduce los tiempos de implementación.

La certificación PCI DSS es más fácil de obtener, PayU captura los datos de las tarjetas de crédito de tus clientes.
