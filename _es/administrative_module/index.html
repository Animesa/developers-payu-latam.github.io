Introducción

El módulo administrativo es el sistema con el que puedes gestionar de forma segura tu cuenta PayU Latam, puedes cambiar tu clave, obtener el reporte de las ventas realizadas, encontrar los datos necesarios para integrar tu página web con nuestra plataforma, usar herramientas para recibir pagos incluso si no tienes página web o transferir dinero a tu cuenta bancaria.

Para hacer uso del Módulo Administrativo debes ir a www.payulatam.com. En la parte superior encontrarás la opción “Ingresa a tu cuenta”, allí podrás introducir tu respectivo usuario y contraseña, los cuales son suministrados vía e-mail en el momento de la creación de tu cuenta en PayU Latam.
¿Aún no tienes una cuenta en PayU Latam? Puedes abrir tu cuenta aquí.
 
Nota: Si olvidaste tu contraseña, debes dar clic en la opción "Olvidé mi clave” e ingresar el correo electrónico con el cual creaste tu cuenta en PayU Latam. El sistema automáticamente enviará a tu correo electrónico un mensaje y el enlace para la activación de tu nueva contraseña.
 

En el Módulo Administrativo de PayU Latam puedes:

  • Ver el dinero disponible en tu cuenta PayU y realizar la transferencia a tu cuenta bancaria.
  • Consultar los movimientos de tus cuentas, generar extractos y reportes.
  • Consultar el estado de las ventas realizadas.
  • Encontrar las credenciales de autenticación para realizar las integraciones: Api Key, Api Login y merchant ID (Id Comercio).
  • Actualizar tus datos bancarios.
  • Configurar el logo de tu empresa para que aparezca en el Web Checkout (pasarela de pagos).
  • Configurar las URLs de tu página de confirmación y página de respuesta (si tienes una integración por API o Web checkout).
