QUERIES WITH API  
PayU provides an API that allows you to check the status of transactions that are generated from placed orders.  
If you want to integrate with our API you must target the requests to the following URLs:  
  • API
Tests: https://stg.api.payulatam.com/reports-api/4.0/service.cgi
Production: https://api.payulatam.com/reports-api/4.0/service.cgi
 
Consider:
  • You must use the API to query when transactions remain in pending status or PENDING.
  • You must perform the first query 7 minutes after the transaction was created.
  • Please don’t retry the query of an order or transaction if the system indicates that it does not exist.
  • For cash payment methods, you can perform only one query per hour.
  • Transactions with payment means such as credit card or bank transfer are available only once every 10 minutes
 
The methods available for the query API are: PING, QUERY ORDER BY IDENTIFIER, QUERY ORDER BY REFERENCE and TRANSACTION RESPONSE QUERY. Below is a description of each.  
PING
 
You can use the doPing method to verify connectivity to our platform.
  • JSON
  • XML
Request example: 
{
  "test": false,
  "language": "en",
  "command": "PING",
  "merchant":
  {
    "apiLogin": "11959c415b33d0c",
    "apiKey": "6u39nqhq8ftd0hlvnjfs66eh8c"
  }
}
 
Response example: 
{
  "code": "SUCCESS",
  "error": null,
  "transactionResponse": null
}
Request example: 
<request>
<language>en</language>
<command>PING</command>
<merchant>
  <apiLogin>11959c415b33d0c</apiLogin>
  <apiKey>6u39nqhq8ftd0hlvnjfs66eh8c</apiKey>
</merchant>
<isTest>false</isTest>
</request>
 
Response example: 
<paymentResponse>
<code>SUCCESS</code>
</paymentResponse>

TRANSACTION RESPONSE QUERY
 
You can use the doTransactionResponseReporting method to query a transaction response.
  • JSON
  • XML
Request example: 
{
   "test":false,
   "language":"en",
   "command":"TRANSACTION_RESPONSE_DETAIL",
   "merchant":{
      "apiLogin":"403ba744e9827f3",
      "apiKey":"676k86ks53la6tni6clgd30jf6"
   },
   "details":{
      "transactionId":"588e3092-6f11-4b77-badf-55695eaa8249"
   }
}
 
Response example: 
{
   "code":"SUCCESS",
   "error":null,
   "result":{
      "payload":{
         "state":"DECLINED",
         "paymentNetworkResponseCode":null,
         "paymentNetworkResponseErrorMessage":null,
         "trazabilityCode":null,
         "authorizationCode":null,
         "pendingReason":null,
         "responseCode":"ENTITY_DECLINED",
         "errorCode":null,
         "responseMessage":null,
         "transactionDate":null,
         "transactionTime":null,
         "operationDate":null,
         "extraParameters":null
      }
   }
}

<request>
   <language>en</language>
   <command>TRANSACTION_RESPONSE_DETAIL</command>
   <merchant>
      <apiLogin>11959c415b33d0c</apiLogin>
