DRUPAL

Requirements:
  • Have an active account in PayU Latam.
  • Have installed Drupal Commerce Kickstart 7.xx
  • Access to Drupal instalation folders.
  • Access to Drupal Backend.
 
Installation:  
 
01. Download the PayU Plugin for Drupal Commerce Kickstart.
.
Download plugin
02. Unzip the file commerce_payulatam_1.0.zip in the path “/sites/all/modules” located in the root of Drupal installation. The uncompressed folder will have an structure like shown in the image.
 
 
Configuration:  
 
01. Enter the Drupal administrator and go to Site settings > Advaced settings > Modules.
02. Find y enable the module “Payment Payulatam”. You can use the search filter to find it easier.

Make sure to save the settings.
03 In the Drupal administrator, go to Store settings > Payment Method. Inside the “Disabled payment method rules” list you will find “Payment gateway PayuLatam”.
04. Click on “enable” and confirm the activation of the module.
05. If the activation of the payment method is successful you will find “Payment gateway PayuLatam” inside the group “Enabled payment method rules”, as shown in the image.
06. Click on “edit”.
07. Find the section “Actions” and click the “edit” button which corresponds to the action “Enable payment method: Payment gateway PayuLatam”. Now you must setup the plugin.

Testing statuses
Once you are in the credit card payment form, and ensuring that you have the message at the top of the gateway "Transaction in test mode", you must:

  1. Enter the text APPROVED in the Full Name field if you want the transaction to be approved, REJECTED if you want it to be rejected or PENDING if you want it to be pending.
  2. You must enter a valid card number in the Card Number field and which corresponds to the selected franchise. For this you can find an online credit card generator.
  3. All other fields can be random.
