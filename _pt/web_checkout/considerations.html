Colômbia  
Formas de pagamento  
 
Required variables
  1. CUS: É uma variável enviada na página de resposta; código único de acompanhamento, é a referência do pagamento dentro do Banco, aplicável somente para pagamentos com o PSE.
  2. pseBank: É uma variável enviada na página de resposta; é o nome do banco, aplicável somente para pagamentos com o PSE.
  3. pseCycle: É uma variável enviada na página de resposta, é o Identificador gerado pelo PSE.
  4. pseReference1 / pseReference2 / pseReference3:: São variáveis enviadas na página de resposta, são os números de referências para pagamentos com o PSE.

Formas de pagamento  
 
Variáveis requeridas
  1. action_code_description: É uma variável enviada na página de resposta; é a descrição do código de resposta do VISANET.
  2. cc_holder: É uma variável enviada na página de resposta; é o nome do cartão do proprietário para a VISANET.
  3. cc_number: Número do cartão de crédito enviado para a VISANET.
  4. processing_date_time: É uma variável enviada na página de resposta; Data e hora do processamento da venda para a VISANET.
  5. request_number: É uma variável enviada na página de resposta; é o número de ordem mais o identificador da transação para a VISANET.
