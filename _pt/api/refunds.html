  • Solicitar reembolso
  • Consultar resposta
{% include sidemenu_api_pt.html %}
 
 
Esta funcionalidade permite solicitar o cancelamento e reembolso de transações que foram pagas com cartão de crédito; e que por motivos de insatisfação do comprador, múltiplos encargos na mesma compra, ou quando o comprador não recebeu o produto ou serviço, requerem a devolução do dinheiro.

Você poderá realizar a solicitação de reembolso empregando o método Refund ou Void, que se encontra disponível para este fim. Uma vez realizada a solicitação, um agente de serviço ao cliente PayU se encarregará de validar a informação e a justificativa do pedido para aprovar a solicitação e realizar o reembolso ou rejeitar a solicitação caso não cumpra os requisitos.
Adicionalmente, você poderá consultar o status da solicitação de reembolso empregando API de Consultas disponível para tal fim.

Considerar:
  • Somente poderão ser realizadas solicitações de reembolso para transações com Autorização e captura com status Capturada realizadas com cartão de crédito. Caso a solicitação se refira a uma forma de pagamento diferente (formas de pagamento em dinheiro, transferência bancária), a solicitação será Rejeitada pelo agente de serviço ao cliente PayU.

  • O sistema somente criará uma solicitação para cada pedido de reembolso, se enviar várias vezes o pedido de uma mesma transação, o sistema somente registrará uma solicitação no módulo e indicará que o pedido já se encontra registrado.

  • Se uma ordem ou transação já foi reembolsada, o sistema indicará que não é possível realizar novamente o pedido. PayU somente aceita solicitações de reembolso de transações que se encontram no status Capturada

  • Se uma solicitação de reembolso de uma ordem ou transação foi previamente Rejeitada, você poderá fazer nova solicitação de reembolso se assim considerar necessário.

  • Uma vez realizada a solicitação o montante da transação passa a ser parte do Saldo Congelado de sua conta PayU Se a solicitação de reembolso é Aprovada, o valor da transação será debitado do Saldo Congelado, já que é reembolsado ao pagador. Se a solicitação de reembolso é Rejeitada, o valor da transação será liberado do Saldo Congelado e retornará para o Saldo disponível de sua conta PayU.
 
 
Solicitar reembolso  

Se uma transação está com status “Capturada”, você poderá utilizar o método Refund para realizar a solicitação de reembolso.

Você deve enviar de maneira obrigatória os seguintes parâmetros:
  • 1. ID da ordem.
  • 2. ID da transação.
  • 3. O tipo de solicitação que será REFUND.
  • 4. Razão para solicitar o reembolso ou cancelamento da transação.
Exemplo pedido:
  • JSON
  • XML
{
   "language": "es",
   "command": "SUBMIT_TRANSACTION",
   "merchant": {
      "apiKey": "P4yU*********L4t4mC0l",
      "apiLogin": "CoM3r*******c1o5"
   },
   "transaction": {
      "order": {
         "id": "45678901"
      },
      "type": "REFUND",
      "reason": "Reason for requesting the refund or cancellation of the transaction",
      "parentTransactionId": "8795a432-12ab-34cd-1234-a9876bc54321"
   },
   "test": false
}

Consultar resposta  

 
Consultar por meio do Módulo Administrativo do PayU.

Para realizar a consulta por meio do módulo administrativo, você deve inserir seu nome de usuário e senha, acessar a aba Relatórios, acessar no menu lateral Ordens, e inserir o ID da Ordem que se deseja consultar nos filtros de busca.
O sistema apresentará a informação da ordem, e mostrará as transações associadas. Uma vez a solicitação de reembolso seja aprovada ou rejeitada, na ordem, será mostrado um novo registro tipo Reembolso com seu respectivo status, seja Aprovado ou Rejeitado, de acordo com a solução dada pelo agente de Serviço ao cliente.

Para executar a consulta de sua solicitação de cancelamento por meio do API, você deve enviar o seguinte pedido:

  • XML

   },

   }
}

Existem três possíveis respostas para sua solicitação de reembolso:
  • Solicitação não resolvida
  • Solicitação aprovada
  • Solicitação rejeitada
