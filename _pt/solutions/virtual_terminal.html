O produto tokenização permite que você armazene os dados dos cartões de crédito dos seus clientes de forma segura, por meio da criação de um token com o qual você poderá realizar cobranças periodicamente. Para o uso das funcionalidades do produto de tokenização desde o Módulo administrativo, você deve contar com um usuário que tenha associado um perfil com as permissões “Token management” a partir do qual você terá acesso para a aba no módulo administrativo “Cobranças com token”.
 
¿Cómo funciona?  
O produto “Tokenização” inclui dois processos independentes:

 
 

As funcionalidades disponíveis para o produto tokenização são:

  • Cadastrar um cartão de crédito
  • Cadastrar vários cartóes de crédito
  • Pagamento online
  • Processar varios pagamentos
  • Consulta de token
  • Eliminação de token
 
Registro individual de cartão de crédito  

Esta funcionalidade permitirá que você coloque individualmente os dados do cartão de crédito de um cliente pagador, o sistema criptografa estes dados e gera um código token com o qual você poderá realizar cobranças de forma segura.
Para realizar o processo de tokenização você deverá contar com estes dados:

  • ID pagador (identificador único do pagador em sua loja).
  • Nome e Sobrenome (Dados do titular, devem ser detalhados da maneira como aparecem no cartão de crédito).
  • Bandeira. Ej.: Visa, MasterCard, American Express, etc.
  • Número do cartão de crédito.
  • Data de expiração do cartão de crédito
  • Número de identificação do cliente pagador
 

01. Para registrar um cartão de crédito e obter o"Token", você deve entrar em “Cobranças com token”, e logo em “Registro tarjeta crédito”, detalhar os campos solicitados e clicar no botão enviar.

 

02. 2. O sistema toma a informação detalhada e realiza o processo de registro do cartão de crédito, gerando e devolvendo como resposta o “token”, o número visível do cartão de crédito, a bandeira e o id do pagador.

 
 
Registro masivo de cartão de crédito  
Esta funcionalidade permitirá que você coloque um lote de até 14.000 registros que inclua os dados dos cartões de crédito dos seus clientes, o sistema criptografará estes dados e gerará os códigos token correspondentes, com os quais você poderá realizar cobranças de forma segura.

Para usar esta funcionalidade, você deve criar um arquivo com o formato CSV com as seguintes condições:

  • Cada registro de cartão de crédito para token deve conter os dados para o registro de um cartão de crédito separados por vírgula e na seguinte ordem,
• ID pagador (identificador único do pagador em sua loja)
• Nome e sobrenome do titular (dados do titular, devem ser detalhados da mesma forma).
• Número do cartão de crédito.
• Data de expiração.
• Bandeira. Ex.: Visa, MasterCard, American Express, etc.
• Número de identificação
  • O arquivo não deve ter cabeçalho. Na primeira linha vai o primeiro registro
  • O arquivo deve estar codificado no padrão UTF-8.
  • O arquivo não deve ter mais de 15000 (14000) registros.

01. Para realizar o registro massivo de cartões de crédito e obter o "Token", para cada uma delas, você deve entrar em “Cobranças com token” e logo em “Cadastrar vários cartões”, logo clicar no botão anexar e selecionar o arquivo CSV com os registros de cartões de crédito para tokenizar.

Selecionando o arquivo para anexar

02. Ao anexar um arquivo, o sistema o toma e gera um novo registro de lote em processamento com a seguinte informação: • Número do lote: identificador único do lote gerado pelo sistema a partir de um arquivo de cartões de crédito para tokenização.
• Data de criação: data em que é realizado o upload do arquivo
• Data de processamento: data em que o sistema processa completamente o arquivo.
• Estado: Os lotes podem se encontrar nos seguintes estados: Error, Pending, Processing y Processed

03.Uma vez que o lote seja corretamente processado, você pode consultar os tokens resultantes clicando na coluna detalhe link “Ver” e o sistema te dará um arquivo em formato CSV que contém: • Token gerado para o cartão de crédito
• ID pagador: identificador único do pagador em sua loja.
• Bandeira Ex. visa, MasterCard, American Express, etc.
• Número visível do cartão de crédito
• Estado
• Nome do cartão titular
• Documento do cartão titular
• Data de criação
• ID Loja
• Námero do lote
• Descrição do erro

 
 
Cobrança individual com token  

Esta funcionalidade permite realizar cobranças, utilizando um código Token que foi previamente criado por nosso sistema, e com o qual se armazenaram os dados dos cartões de crédito dos seus clientes de forma segura.

Para esta funcionalidade, você deve contar com os seguintes dados:

  • ID conta: identificador da conta virtual na PayU Latam para qual deve ser associada a transação.
  • Código secuencial de la tarjeta de crédito “Token”
  • Código de segurança: este campo é opcional e somente será necessário no caso de as entidades financeiras nos obriguem a enviá-lo.
  • Número de parcelas: você pode definir desde 1 até 12 parcelas.
  • E-mail do comprador: este dado é opcional, ficará associado á informação da transação e será utilizado para envio de notificação da cobrança realizada.
  • Referência de venda: aplicam as restrições de PayU Latam para lojas que façam esse esquema de validação de refer≖ncia única.
  • Descrição da venda.
  • Moeda: codificação ISO da moeda para utilizar na transação.
  • Valor Total: deve incluir o imposto.
  • Imposto.
  • Base de devolução do imposto.
  • Valor adicional.
  • Idioma: se toma na conta para o envio das notificações no processo de compra.
 

01. Para realizar uma cobrança online, utilizando o Token de um cartão de crédito, você deve colocar “Cobranças com token” e logo “Pagamento online”, detalhar os campos solicitados e clicar no botão enviar.

Consulta de token  

Esta funcionalidade permite realizar a consulta dos Token de cartões de crédito que você tem registrado

 
  • Id pagador: identificador único do pagador em sua loja.
  • Código sequencial do cartão de crédito “Token”
  • Ranking de datas (data inicial – data final)

01. 1. Para realizar a consulta de um ou vários “Token” deves ingressar “Cobros con token” e logo a “Gestão de token”, definir os parâmetros de busca desejados e clicar no botão consultar.

02. Ao consultar esse sistema retornará uma lista de todos os registros encontrados com a seguinte informação: • Data de registro.
• ID pagador.
• Código sequencial do cartão de crédito Token.
• Número visível do cartão de crédito
• Franquicia.

 
 
Eliminar token  
Esta funcionalidade permite que você elimine um registro de cartão de crédito tokenizada, de tal maneira que não será possível voltar a executar cobranças utilizando o código sequencialToken gerado.

Para eliminar um token os dados solicitados são:

  • Id pagador: identificador único do pagador em sua loja.
  • Código sequencial de cartão de crédito “Token”.
 

01. Você deve entram em “Cobranças com token” e logo a "Gestão de token”, você deve realizar a consulta do token que você deseja eliminar e ao publicá-lo colocá-lo nos resultados da consulta, clicar na coluna opções no link “Dar de baja”.

02. Ao clicar sobre o link “Dar de baja”, ”, o sistema solicitará sua confirmação para eliminação do token.

03. Ao aceitar a ação, o sistema confirma para o usuário que o cartão de crédito tenha sido desassociado.
