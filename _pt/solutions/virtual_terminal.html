Terminal virtual

 
Com este produto, você pode receber pagamentos de forma presencial ou por telefone.

Seu cliente forneceria os dados necessários para realizar a transação e poderá receber uma confirmação da compra imediatamente.

Não necessitas contar com uma página web para receber pagamentos de forma segura

Nota:
  • Este produto esta temporariamente disponível só para contas do Colômbia, México e Brasil.
  • Se você deseja ativar este produto para sua conta, você deve escrever para sac@payulatam.com.
 
Para realizar cobranças você deve:  

01. Insira a http://www.payulatam.com/, na parte superior você encontrará a opção
“Insira sua conta”, ali você poderá introduzir seu respectivo nome de usuário e senha, os quais são fornecidos via e-mail no momento da criação de sua conta no PayU Latam.

02. Para realizar cobranças você deve entrar na aba Ferramentas na opção de “Processamento em linha”.

03. Você deverá colocar os dados solicitados para realizar a cobrança. Os campos podem variar de acordo com o país e ao meio de pagamento empregado..

04. Ao terminar de completar o formulário você verá os dados inseridos de maneira consolidada.

05. Uma vez que você envie os dados de cobrança, você receberá uma notificação do sistema com o resultado da transação, adicionalmente seu cliente receberá um e-mail com a informação de pagamento.
